## ПОСТАНОВКА ЗАДАЧИ:

Конечной целью является построение GraphRAG системы, базой знаний для которой будут выступать научные статьи arxiv, посвященные теме "LLM for Time Series Forecasting". 

Ниже представлена онтология для анализа и извлечения знаний из этих статей, которая позволяет выделить классы и отношения между ними - это нужно сделать путём анализа материалов статей для базы знаний.

Цель извлечения знаний из научных статей - определить постановки задач, виды моделируемых и прогнозируемых временных рядов, способы применения LLM для прогнозирования временных рядов, виды используемых LLM, способы подготовки данных, метрики для оценки эффективности, бенчмарки и оценки эффективности отдельных подходов и методов.
