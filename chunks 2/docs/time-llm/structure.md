## Примечания

1. **Рисунки (Figure 1–10)** в markdown-версии представлены текстовыми описаниями. Не содержат retrieval-данных.
2. **13 типов вопросов (v1.2):** А, А+, В, З, Г, Б, Д, Е, И, Ж, К, Л, М — наиболее полный набор. Старая версия (v1.0) покрывала только А и Д с неверными индексами.
3. **Тип М (q12)** — уникальный для time-llm: завершающий шаг прогноза (Output Projection).
4. **Таблицы результатов** (Табл. 1–5, 10–16) — основной источник для типов Ж, А+. Приложения D–E содержат развёрнутые версии.
5. **Раздел 4.5 Model Analysis** — многоцелевой: А+ (абляции), Б (подтверждение улучшений), К (scaling law).
6. **Список литературы** (362–481) не подлежит чанкингу.
